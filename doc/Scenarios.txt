Сценарии (ТЗ)
1. Считывание QR кода:
   1.1 Пользователь входит в приложение, и появляется страница, предлагающая считать QR-code.
   1.2 Пользователь нажимает на кнопку подтверждения и переходит в режим считывания QR кода.
   1.3 После считывания приложение открывает страницу создания заказа.

2. Вызвать официанта:
   2.1 Пользователь нажимает кнопку вызова официанта.
   2.2 Приложение блокирует кнопку на какое-то количество времени.
   
3. Перейти на сайт заведения:
   3.1  Пользователь нажимает на кнопку перехода на сайт кафе.
   3.2  Приложение запускает браузер и открывает вкладку этого кафе.
     
4. Сделать заказ:
   4.1 Пользователь вводит количество гостей.
   4.2 Открывается вкладка меню, где пользователь выбирает необходимые ему пункты и для каждого из них указывает количество порций.
   4.3 Когда пользователь определился с заказом, он нажимает на кнопку подтверждения, переходя к просмотру чека.
   4.4* Пользователь может по желанию оставить комментарий к заказу для указания определенных особенностей заказа (пр.: без помидоров, больше салфеток).
   4.5 После того как пользователь убедился в правильности своего заказа, он нажимает на кнопку “опатить” и переходит на страницу оплаты.
